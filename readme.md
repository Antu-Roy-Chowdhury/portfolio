# Antu Roy Chowdhury - Portfolio Website

A modern, responsive portfolio website built with Next.js, React, and Tailwind CSS. This portfolio showcases skills, projects, research, and achievements with a minimalist aesthetic and creative interactive elements.

## Features

- **Modern Design**: Clean, minimalist aesthetic with a black and neon blue color scheme
- **Interactive Elements**: Custom cursor, animated background, hover effects
- **Responsive Layout**: Fully responsive design that works on all device sizes
- **Multi-page Structure**: Separate pages for Home, About, Projects, Skills, Research, and Contact
- **Creative Components**: Glass morphism, gradient text, timeline layouts
- **Performance Optimized**: Fast loading and smooth animations

## Pages

- **Home**: Introduction and section highlights
- **About**: Personal story, experience timeline, education, and certifications
- **Projects**: Showcase of development and research projects
- **Skills**: Technical and soft skills with creative visualization
- **Research**: Publications, current research, and methodology
- **Contact**: Contact form and information

## Technologies Used

- **Next.js**: React framework for production
- **React**: JavaScript library for building user interfaces
- **Tailwind CSS**: Utility-first CSS framework
- **Framer Motion**: Animation library for React
- **Lucide React**: Icon library

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git
   cd portfolio-website
```

2. Install dependencies:

```shellscript
npm install
# or
yarn install
```


3. Run the development server:

```shellscript
npm run dev
# or
yarn dev
```


4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.


## Customization

### Changing Content

- Update personal information in the component files
- Replace placeholder images with your own
- Modify project details in the `Projects.jsx` component


### Styling

- Main color scheme is defined in `globals.css`
- Tailwind configuration can be modified in `tailwind.config.js`


## Deployment

This portfolio can be easily deployed on Vercel:

1. Push your code to a GitHub repository
2. Import the project in Vercel
3. Deploy
